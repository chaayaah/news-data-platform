# News Data Platform

## Overview

A modern Data Engineering platform inspired by real-world content ingestion
pipelines.

The platform ingests XML/JSON feeds, performs configuration-driven schema
mapping, validates data quality, implements a Bronze/Silver/Gold architecture,
and publishes curated data through APIs, dashboards, and machine learning
consumers.

## Tech Stack

- Python
- PySpark
- PostgreSQL
- FastAPI
- Streamlit
- Kafka
- MLflow
- Docker
- GitHub Actions
